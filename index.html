healthierU

Full-Stack Group Project

Click me!

The group I worked with set out to build an app that allowed both coaches and athletes to create accounts, log in, and interact with each other. We developed the ability for athletes to select which coach they wanted to be assigned to, which then allowed their selected coach to assign the athlete that selected them workouts.

We did this using Postgres, Express, Node.js, Sequelize.js, HTML5, CSS, and JavaScript. My role mainly consisted of setting the up the back end, which involved setting up our database and correctly setting up table relations and ensuring that routes worked as needed and allowed the user to navigate the site with ease.

This project taught me the importance of a well thought out strategy when approaching a project. Many of our issues came down to not knowing exactly where we wanted the project to go, but once we figured those issues out as a team it became much easier to build the site and know what we wanted the code to do.
